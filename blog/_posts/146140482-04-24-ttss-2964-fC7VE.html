In which I get annoyed at No-IP and move to DuckDNS...I have used No-IP as my dynamic DNS provider since DynDNS started charging. A friend recently pointed me in the direction of DuckDNS which doesn't require me to click a link every month to maintain the service. Here's how to set it up on FreeNAS:

  1. Sign up for a DuckDNS account.
  2. Make a new subdomain.
  3. Get your token (a very long sequence of random characters on the home page)
  4. Login to your Freenas server GUI
  5. Go to System->Cron Jobs->Add New Cron Job
  6. Set up the new cron job.
    1. Set it run as 'nobody

    3. The command to run is: /usr/local/bin/curl https://duckdns.org/update/subdomain/token
